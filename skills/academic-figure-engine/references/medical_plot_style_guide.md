# 医学期刊统计图风格指南

> 结构模板来自 Google PaperBanana NeurIPS 2025 Style Guide，
> 内容适配医学期刊（EJHF, Circulation, JAMA, Lancet, BMJ 等）。
> 融合 publication_guidelines.md 中的技术规范。

---

## 1. 整体美学定位

**关键词：临床精确、数据优先、保守优雅**

- **背景**：严格白底（`#FFFFFF`），不使用灰色/彩色背景
- **信息密度**：中等——数据清晰可读，但不过度留白
- **色彩克制**：灰白为基调，仅在信息编码时使用有限色彩（通常 ≤4 色）
- **可及性**：必须通过色盲模拟（Okabe-Ito 或 Tol 系列）

与 CS 会议风格的核心区别：
| 维度 | CS 会议 (NeurIPS) | 医学期刊 |
|------|-------------------|---------|
| 审美取向 | 现代、有设计感 | 权威、机构化 |
| 背景 | 白色或浅灰均可 | 严格白底 |
| 色彩 | 柔和粉彩、多色 | 灰度+1-2个强调色 |
| 网格 | 虚线或无均可 | 极简或无 |
| 轴线 | 四面框或开放均可 | 开放式（去掉顶右）优先 |

## 2. 色彩方案

### 分类数据（组间比较）

**首选方案：Okabe-Ito 色盲友好色板**
```python
OKABE_ITO = ['#E69F00', '#56B4E9', '#009E73', '#F0E442',
             '#0072B2', '#D55E00', '#CC79A7', '#999999']
```

**备选方案：Tol Bright**
```python
TOL_BRIGHT = ['#4477AA', '#EE6677', '#228833', '#CCBB44',
              '#66CCEE', '#AA3377', '#BBBBBB']
```

**医学专用约定**：
- 对照组/参考组：深蓝灰 `#2C3E50` 或灰色
- 治疗组/暴露组：红/橙系 `#E74C3C` 或 `#D55E00`
- 第三组（如有）：绿/青 `#27AE60` 或 `#009E73`
- 避免纯红+纯绿组合（红绿色盲不可区分）

### 连续数据（热图/梯度）

- **单向**：`viridis`（蓝→黄）或 `plasma`（紫→橙），感知均匀
- **双向**：`RdBu_r`（红蓝发散），center=0 用于相关矩阵
- **禁用**：`jet`/`rainbow`（过时且感知不均匀）

### 统计显著性颜色编码

- **p<0.05**：可用红色标注（`#E74C3C`）
- **p≥0.05**：灰色或无特殊标记
- 避免过度使用星号（*）标注——直接写 p 值更精确

## 3. 轴线与网格

### 轴线 (Spines)
- **首选**：开放式（去掉 top + right spine）
- **备选**：四面框（某些期刊偏好，如 Circulation）
- 轴线颜色：黑色或深灰 `#333333`
- 轴线宽度：0.8pt

```python
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)
ax.spines['bottom'].set_linewidth(0.8)
ax.spines['left'].set_linewidth(0.8)
```

### 网格 (Grid)
- **首选**：无网格（数据自己说话）
- **备选**：浅灰虚线（仅 Y 轴方向），alpha=0.3
- 网格永远在数据后面（zorder=0）

```python
# 如需网格
ax.yaxis.grid(True, linestyle='--', alpha=0.3, color='#CCCCCC', zorder=0)
ax.xaxis.grid(False)
```

### 刻度 (Ticks)
- 方向：朝内或无（不朝外）
- 字号：≥6pt（最终打印尺寸）
- X 轴标签：优先水平；如过长则 45° 旋转
- 避免科学计数法（除非数值跨度极大）

## 4. 排版 (Typography)

- **字体族**：Arial（医学期刊通用）；备选 Helvetica
- **轴标签**：9pt，sentence case，含单位括号
- **刻度标签**：8pt
- **图例**：8pt
- **面板标签**：10pt Bold（A, B, C），位于子图左上角
- **注释**：8-9pt，斜体用于统计量（如 *p* < 0.001）

### 标注策略
- **直接标注优于图例**：如果线条/柱条 ≤4 组，优先在线/柱旁直接标注名称
- **KM 曲线**：Number at risk 表格放在图下方
- **Forest plot**：HR(95%CI) 和 p 值放在右侧数值列
- **避免**：图上写标题（标题在 figure caption 中）

## 5. 图类型专项指南

### Kaplan-Meier 生存曲线

| 要素 | 规范 |
|------|------|
| 线型 | 阶梯函数 (`step`)，线宽 1.5-2pt |
| 95% CI | 半透明阴影带（alpha=0.15-0.2） |
| 删失标记 | 短竖线 tick（`+` 形），大小适中 |
| Number at risk | 图下方表格，与 X 轴刻度对齐 |
| Log-rank p | 图内右上角或左上角注释 |
| 中位生存 | 虚线标注（如适用） |
| 图例位置 | 右下角或下方（不盖数据） |
| Y 轴范围 | 0-1.0 或 0-100% |

### Cox Forest Plot

| 要素 | 规范 |
|------|------|
| 参考线 | HR=1 处灰色虚线 |
| X 轴 | 对数刻度 |
| 方块大小 | ∝ 权重（事件数或样本量的倒数） |
| 数值列（右侧） | HR (95% CI), p-value |
| 事件列（左侧） | Events/Total per group |
| 模型标注 | Model 1 (unadjusted), Model 2 (adjusted) 等 |
| CI 线宽 | 1.0-1.5pt |

### 热图 (Heatmap)

| 要素 | 规范 |
|------|------|
| Colormap | 双向：`RdBu_r` (center=0)；单向：`viridis` |
| 单元格 | 正方形，白色细边 (linewidths=0.5) |
| 数值标注 | 每个 cell 内标注值（fmt='.1f' 或 '.2f'） |
| Colorbar | 右侧，有标签和单位 |
| 行列标签 | 完整可读（列标签 45° 旋转 if 过长） |
| 标准化方法 | 在 caption 或注释中说明 |

### 箱线图 / 小提琴图

| 要素 | 规范 |
|------|------|
| 中位线 | 粗线标注 |
| 均值 | 可选用菱形/三角标注 |
| 异常值 | 小点（不要大圆圈） |
| 组间比较 | 括线 + 星号或 p 值 |
| 填充 | 半透明组色（alpha=0.6-0.8） |
| 小提琴 | 如用，内部加箱线图 |

### 散点图

| 要素 | 规范 |
|------|------|
| 点大小 | 统一 15-30，或按权重缩放 |
| 回归线 | 实线 + 95% CI 阴影带 |
| 相关系数 | 图内注释 R²/rho + p 值 |
| 多组 | 不同颜色+不同标记形状（○ △ □ ◇） |
| 密度区 | 如数据量大，用透明度或 hexbin 替代逐点 |

## 6. 常见陷阱（必须避免）

| # | 陷阱 | 后果 | 正确做法 |
|---|------|------|---------|
| 1 | jet/rainbow colormap | 审稿人一票否决 | 用 viridis/RdBu_r |
| 2 | 图上写标题 | 与 caption 冲突 | 标题只在 caption |
| 3 | 3D 柱状图 | 视觉误导 | 用 2D 柱状图 |
| 4 | 双 Y 轴 | 极易误读 | 拆分为两张图 |
| 5 | 红+绿区分组 | 色盲不可读 | 用蓝+橙或形状区分 |
| 6 | 线图无数据点标记 | 无法精确定位值 | 加 marker (○□△◇) |
| 7 | Y 轴不从 0 | 夸大差异 | 从 0 开始或注明截断 |
| 8 | 实心黑色网格线 | 干扰数据阅读 | 浅灰虚线或无网格 |
